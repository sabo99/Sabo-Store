# Sabo-Store

**E-Commerce Gear Computer** Android Java (<a href="https://firebase.google.com">**`Firebase`**</a> + <a href="https://square.github.io/retrofit/">**`Retrofit`**</a> + <a href="https://developer.android.com/training/data-storage/room">**`Room DB`**</a>)

Database menggunakan <a href="https://firebase.google.com">**`Firebase`**</a> dengan IDE <a href="https://developer.android.com/studio">**`Android Studio`**</a> dan bahasa pemrogramman Java 

### **Fitur Aplikasi E-Commerce Sabo Store** :

- **Firebase**
  - `Collection / Database Real Time`
  - `Change Email Verification via Gmail`
  - `Forgot Password via Gmail`
  - `Delete Account`
  - `Upload image / Change image profile`
  - `Order History`
- **Room DB**
  - `Cart Items`
  - `Favorite Items`
- _**SweetAlertDialog**_
  - `Modal Dialog with Custom Content View`
  - `Message Dialog`
  - `Loading Dialog`
